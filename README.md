# Pt. I. - Let's Get Dapper

1. When in doubt, `yarn` it out:
  Run `yarn`

2. Run `yarn start` to start the client.  This should open up `localhost:3000` in a chrome browser.

You should see the Dapper UI with no smart contracts loaded.
